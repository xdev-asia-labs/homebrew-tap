# xdev-asia-labs Homebrew Tap

This is the official Homebrew tap for xdev-asia-labs tools.

## Installation

```bash
brew tap xdev-asia-labs/tap
brew install xexplain
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `xexplain` | System Intelligence CLI for macOS |
